  1. Python 依赖：安装 pandas

打开开始菜单 → Anaconda3 → Anaconda Prompt， 并输入 conda install pandas 回车执行， 如下图所示。

﻿﻿


输入 y 确认安装，安装完毕后关闭该窗口。苹果电脑打开终端，输入 conda install pandas 回车执行，然后输入 y 确认安装即可。


（以下都是同样步骤，只是输入的不同）


  1. Python 依赖：安装 matplotlib 和 seaborn

安装的方式和之前一样，我们打开开始菜单 → Anaconda3 → Anaconda Prompt。 输入 conda install matplotlib 并回车。输入 y 确认，完成安装。


接下来，我们安装 seaborn 工具包，继续输入 conda install seaborn 并回车，并在如下界面中输入 y 回车，确认安装。


安装完毕后，关闭命令行窗口。


  1. Python 依赖：安装 wordcloud，jieba

打开开始菜单 → Anaconda3 → Anaconda Prompt， 并输入 conda install -c conda-forge wordcloud 回车执行。输入 y 确认安装，安装完毕后关闭该窗口。


打开开始菜单 → Anaconda3 → Anaconda Prompt， 并输入 conda install -c conda-forge jieba 回车执行。输入 y 确认安装，安装完毕后关闭该窗口。


  1. Python 依赖：安装opencv、pillow

打开开始菜单 → Anaconda3 → Anaconda Prompt， 并输入 conda install opencv

回车执行。输入 y 确认安装，安装完毕后关闭该窗口。


打开开始菜单 → Anaconda3 → Anaconda Prompt， 并输入 conda install pillow

回车执行。输入 y 确认安装，安装完毕后关闭该窗口。
